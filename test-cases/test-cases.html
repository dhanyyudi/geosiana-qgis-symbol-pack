Geosiana QGIS Pack V2 Test Cases

Preview ini khusus untuk paket siap-upload. Semua path sudah disesuaikan ke struktur folder paket, sehingga folder ini bisa langsung dipindah ke repo lain tanpa perlu mengedit referensi file lagi.

Point

SVG marker tunggal untuk layer titik.

  • svg/geosiana-marker_extra/u0032-two.svg
  • Gunakan sebagai SVG marker langsung pada layer titik.
  • Cocok untuk POI, lokasi dokumentasi, atau titik fasilitas.

Line

Marker line memakai SVG berulang di atas garis utama.

  • svg/geosiana-marker_default/u007D-braceright.svg
  • Gunakan pada Marker line, aktifkan rotate marker to follow line direction.
  • Cocok untuk sungai, arus, jalur evakuasi, atau arah sirkulasi.

Polygon

SVG fill atau point pattern fill untuk area.

  • svg/geosiana-marker_forestry/u0042-b.svg
  • Gunakan pada SVG fill atau Point pattern fill untuk polygon.
  • Cocok untuk kawasan vegetasi, ruang terbuka hijau, atau area tematik.
