# Geosiana Curated Pack

Paket ini ditujukan untuk import cepat ke `Style Manager` QGIS dan memuat subset simbol yang lebih mudah dicari berdasarkan kategori tematik.

## File Pack

- `style-packs/geosiana-full-embedded.xml`: semua simbol marker hasil ekspor, dengan nama semantik untuk glyph yang sudah dipetakan.
- `style-packs/geosiana-curated-v2-embedded.xml`: subset simbol terpilih dengan kategori operasional seperti `batas-administrasi`, `jalan`, dan `vegetasi`.
- `style-packs/geosiana-line-fill-starter.xml`: starter pack untuk line dan polygon berbasis recipe legenda LapakGIS.

## Catatan Penggunaan

- Tag `untuk-point` dipakai untuk ikon yang paling cocok langsung sebagai marker point.
- Tag `untuk-line` dipakai untuk ikon yang disarankan sebagai sub-symbol pada `Marker line`.
- Tag `untuk-polygon` dipakai untuk ikon yang cocok pada `SVG fill` atau `Point pattern fill`.
- Category `batas-administrasi` dan `jalan` sengaja diisi marker yang paling masuk akal untuk dekorator line symbol, bukan pengganti line style native QGIS sepenuhnya.
